# Phonebook Project

Este é um projeto de um aplicativo de agenda telefônica.

## Descrição
O aplicativo permite que os usuários armazenem e gerenciem contatos telefônicos de forma simples e eficiente.

## Funcionalidades
- Adicionar novos contatos
- Editar contatos existentes
- Remover contatos
- Visualizar todos os contatos

## Link do Backend
Você pode acessar o backend do projeto através do seguinte link: [Phonebook Backend](https://phonebook-backend-hdnm.onrender.com)

## Como Executar o Projeto
1. Clone o repositório.
2. Instale as dependências usando `npm install`.
3. Execute o aplicativo com `node index.js`.
